# xfce4-moc-plugin

**xfce4-moc-plugin** позволяет взаимодействовать с консольным аудиоплеером MOC через xfce-panel

### Установка

    % cd xfce4-moc-plugin
    % ./autogen.sh --prefix=/usr
    % make
    % sudo make install
